Acerca de la Clase
El propósito de esta página web es el de servir como portafolio para las actividades realizadas en la clase de Software Architecture AKA Cloud Cloud Computing.
¿Qué es la arquitectura de nube?
La arquitectura de nube hace referencia a la forma en la que se integran las distintas tecnologías para crear las nubes, es decir, los entornos de TI que separan, agrupan y comparten los recursos flexibles a través de la red. La arquitectura define la manera en la que se conectan todos los elementos y las funciones necesarios para diseñar la nube: la plataforma en línea donde se pueden ejecutar las aplicaciones.
¿Qué espero aprender del curso?
Para el final del curso, espero aprender de la clase una serie de estrategias que me permitan, analizar y diseñar sistemas distribuidos en la nube. Para de esta manera poder tomar decisiones informadas a la hora de realizar decisiones de arquitectura, además de diseñar sistemas estables con mecanismos de deploy automáticos que permitan identificar errores y manejarlos.
